# create-issue

Create GitHub issues from natural language descriptions directly from Claude Code.

## Installation

```
claude plugin install spadarshut/spadarskills/create-issue
```

## Usage

```
/create-issue <description>
```

## Modes

- **Minimal** — Uses your description as-is to create the issue immediately, no questions asked.
- **Extensive** — Asks clarifying questions, explores the codebase for context, and composes a rich, implementation-ready issue description.
